Custom CPU Script

Kode di bawah ini adalah sebuah skrip shell untuk mengatur pengaturan CPU pada perangkat Android yang menggunakan ROM kustom. Skrip ini memungkinkan pengguna untuk mengatur frekuensi minimum (MINFREQ) dan maksimum (MAXFREQ) CPU, serta mengubah governor yang mengontrol bagaimana CPU beroperasi. 

/sys/devices/system/cpu/cpu0/cpufreq/scaling_governor
        else
            echo "Please use a valid type of CPU governor!" 1>&2
        fi
    fi
fi

Cara Penggunaan

  1. Clone repository ini dengan menggunakan perintah berikut:
git clone https://github.com/rakarmp/CpuOverclock.git
  1. Setelah berhasil mengklon repository, jalankan skrip dengan perintah berikut:
./customcpu.sh

Fitur-Fitur

  • Mengatur frekuensi minimum (MINFREQ) dan maksimum (MAXFREQ) CPU.
  • Mengubah governor yang mengontrol bagaimana CPU beroperasi.
  • Low Memory Killer
  • Disable Thermal Throttling
  • Disable dynamic memory management
  • Disable dynamic scaling service
  • Set I/O Scheduler => noop
  • Screen animation off
  • OpenGl Optimize
